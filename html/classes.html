CSS Classes

The class= attribute can be used to ensure elements have the same style without repetition.

For example:

    div.cities {
      background-color: black;
      color: white;
      margin: 20px 0 20px 0;
      padding: 20px;
    }

defines a style class to be applied to all <div> elements with the class="cities" attribute.

In-line elements can also use the class= attribute: For example:

    span.note {
      font-size: 120%;
      color: red;
    }

might be used thus:

    <h1>My <span class="note">Important</span> Heading</h1>
    <p>This is some <span class="note">important</span> text.</p>

and would appear as:

My Important Heading

This is some important text.

A style class may be restricted to a particular element type (as above) or may be applied to any element by omitting the type from the definition:

    .note {
      font-size: 120%;
      color: red;
    }
