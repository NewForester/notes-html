HTML Iframes

An <iframe> element is used to display a web page within a web page. It in-lines a web page within a frame.

    <iframe src="URL"></iframe>

the src= attribute specifies the Internet address of the page to in-line, which may be a full URL or a relative file path.

The height= and width= attributes may used to specify the dimensions of the <iframe> element. These are in pixels by default but can also be specified in percent.

An <iframe> element has a border by default. This can be removed or given a different style using the CSS border: property with the style= attribute. For example:

    <iframe src="https://en.wikipedia.org/wiki/" height="200" width="600" style="border:2px solid blue;"></iframe>

An <iframe> element can be the target of a link. Clicking on the link will display the target page in the <iframe> element.

There is an implicit connection between the target= attribute of the link and the name= attribute of the frame.

    <iframe src="https://en.wikipedia.org/wiki/" height="200" width="600" name="gnu_frame"></iframe>

    <p><a href="https://www.gnu.org/" target="gnu_frame">GNU Project Home Page</a></p>

will appear as:
