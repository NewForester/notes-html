HTML Lists

HTML defines three types of list:

  1. unordered lists;
  2. ordered lists;
  3. description lists.

List items can contain other HTML elements, not just text. A nested list is list item that is, itself, a list element.

The <ul> element defines an unordered list. The <li> element defines individual list items.

    <ul>
      <li>Orange</li>
      <li>Apple</li>
      <li>Banana</li>
    </ul>

will appear as:

  • Orange
  • Apple
  • Banana

The list-style-type property may be used to specify an alternative marker for list items.

  1. disc - bullet marker (the default)
  2. circle - circle marker
  3. square - a black square
  4. none - no marker

The <ol> element defines an ordered list. List items are numbered.

    <ol>
      <li>Orange</li>
      <li>Apple</li>
      <li>Banana</li>
    </ol>

will appear as:

  1. Orange
  2. Apple
  3. Banana

The type= attribute may be used to specify alternative numbering for list items.

  1. type="1" - numbers (the default)
  2. type="A" - uppercase letters (A, B, ...)
  3. type="a" - lowercase letters (a, b, ...)
  4. type="I" - uppercase Roman numerals (I, II, ...)
  5. type="i" - lowercase Roman numerals (i, ii, ...)

The <dl> element defines an description list.

It is a list of terms with a description for each. The <dt> and <dd> elements define terms and descriptions respectively.

    <dl>
      <dt>Orange</dt>
      <dd>a citrus fruit</dd>
      <dt>Banana</dt>
      <dd>a tropical fruit</dd>
    </dl>

will appear as:

Orange
a citrus fruit
Banana
a tropical fruit

CCS styles can be used to style lists in many different ways. A common example is to list elements horizontally and turn list items into a selection menu.

    <head>
      <style>
        ul {
            list-style-type: none;
            margin: 0;
            padding: 0;
            overflow: hidden;
            background-color: #333333;
        }

        li {
            float: left;
        }

        li a {
            display: block;
            color: white;
            text-align: center;
            padding: 16px;
            text-decoration: none;
        }

        li a:hover {
            background-color: #111111;
        }
      </style>
    </head>

    <body>
      <ul>
        <li><a href="#home">Home</a></li>
        <li><a href="#news">News</a></li>
        <li><a href="#contact">Contact</a></li>
        <li><a href="#about">About</a></li>
      </ul>
    </body>
