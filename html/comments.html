HTML Comments

HTML comment tags enclose comments in HTML documents.

    <p>This is a
    <!--
    A comment that will not be displayed
    -->
    sentence.</p>

will appear as:

This is a sentence.

Comments are not displayed: they may be used to document the HTML source code.

Comments are useful for debugging: they may be used to comment out parts of the document.

Conditional comments may be browser specific (and not part of the standard).

    <!--[if IE 9]>
        .... some HTML here ....
    <![endif]-->
