HTML Styles - CSS

CSS (Cascading Style Sheets) describe how HTML elements are to be rendered (on screen, paper or other media). It is covered in detail in the CSS Tutorial. Since one style sheet can apply to many elements on many web pages, it can save a lot of work.

CSS styles can be applied to an element:

  1. inline - using the style attribute of a single HTML element
  2. internal - using the <style> element in the <head> section of an HTML document
  3. external - using a CSS style sheet (often across all pages of a web site)

Inline CSS is used to apply a unique style to a single HTML element by using the style= attribute of the element.

    <h3 style="color:blue;">This is a Blue Heading</h3>

will appear as:

A Blue Heading

Internal CSS is used to define a style for a single HTML page using the <style> element in the <head> section of a page.

    <head>
        <style>
            h4      {color: red;}
        </style>
    </head>
    <body>
        <h4>A Red Heading</h4>
    </body>

will appear as:

A Red Heading

An external style sheet is used to define the style for many HTML pages by using the <link> element in the <head> section of a page. The appearance of an entire web site can be defined in a single file.

    <head>
        <link rel="stylesheet" href="css-styles.css" />
    </head>
    <body>
        <h5>A Green Heading</h5>
    </body>

will appear as:

A Green Heading

The style sheet "css-styles.css" will contain (at least):

    h5      {color: green;}

External style sheets can be referenced using a full URL or a relative file path.

HTML Styles and CSS Properties

There are many CSS properties. Here are some of those used most often.

  1. color - defines the text color;
  2. font-family - defines the text font;
  3. font-size - defines the text size;
  4. border - defines a border around an element:
  5. padding - defines a padding (space) between text and the border:
  6. margin - defines a margin (space) outside the border:

The id= attribute may be used to apply a specific style set to one element. The intention is that an element id is unique within a page.

    #p01  {
        color: red;
        font-family: courier;
        font-size: 160%;
    }

Then 

    <p id="p01">I am different</p>

will appear as:

I am different

The class= attribute may be used to apply a specific style set to a set of elements.

    p.surrounded {
        border: 1px solid powderblue;
        padding: 30px;
        margin: 50px;
    }

Then 

    <p class="surrounded">Help ! Get me out of here !</p>

will appear as:

Help ! Get me out of here !
